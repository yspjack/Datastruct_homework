## 填空题

1. 下列程序判断字符串s 是否对称，对称则返回1，否则返回0；如 f("abba")返回1，f("abab")返回0；

    ```C
    int f(char *s)
    {
        int i=0,j=0;
        while (s[j])
        j++;
        for(j--; i<j  && s[i]==s[j]; i++,j--);
        return (i>=j) ;
    }
    ```
