2. 假设n为2的乘幂，并且n>2，试求下列算法的时间复杂度及变量count的值(以n的函数形式表示，在本课程作业和考试范围内，除非特别声明，对数都是以2为底，即 $log_2n$ 可表示为：log n或者log(n)；m*n的形式可表示为mn)。

    ```C
    int time (int n) {
        int count=0, x=2;
        while ( x < n/2 ) {
            x=2*x;
            count++;
        }
        return (count);
    }
    ```
    时间复杂度：`O(log(n))`  count值：`log(n)-2`
